# Portfolio

Personal portfolio built with Next.js and Tailwind CSS.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- **Next.js 15** — React framework
- **Tailwind CSS 4** — styling
- **Portfolio data** — `src/data/portfolio.js`

## Structure

```
app/
├── globals.css       # Global styles + keyframes
├── layout.jsx        # Root layout
└── page.jsx          # Home page
src/
└── data/
    └── portfolio.js  # Portfolio content
```
